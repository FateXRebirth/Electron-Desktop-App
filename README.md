# Electron-Desktop-App

The resaon I create this project is to know how to use Electron to create a desktop application.

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To run electron

```sh
$ npm start
```

### To Package & Build

For Windows

```sh
$ npm run package-win
```

For Mac

```sh
$ npm run package-mac
```

For Linux

```sh
$ npm run package-linux
```
